#  ML_sesion

Этот репозиторий содержит финальный проект по курсу Machine Learning. В проекте реализованы алгоритмы линейной и логистической регрессии "с нуля" (используя `NumPy`), а также проведено сравнение с ансамблевыми методами (`Random Forest`) для предсказания популярности музыкальных треков.

## Описание проекта

Проект решает две задачи машинного обучения на данных Spotify:

### 1. Регрессия (Линейная регрессия)
Предсказание точного балла популярности трека (`track_popularity`) на основе его аудио-характеристик.
* **Реализация:** Собственный класс `LinearRegressionSGD`.
* **Метод обучения:** Mini-batch Gradient Descent (стохастический градиентный спуск по мини-батчам).
* **Метрика оптимизации:** MSE (Mean Squared Error).
* **Результат:** Анализ сходимости алгоритма (график потерь) и визуализация линии регрессии.

### 2. Бинарная Классификация
Определение того, станет ли трек "хитом" (популярность выше медианы по датасету).
* **Модели:**
    1.  **Логистическая регрессия (Custom Logistic Regression):** Реализована вручную с использованием сигмоиды и минимизации Log-Loss функции.
    2.  **Random Forest Classifier:** Использована реализация из `scikit-learn` для сравнения качества.
* **Метрики:** Accuracy, Precision, Recall, F1-Score, Confusion Matrix.
* **Визуализация:** Сравнение ROC-кривых и расчет ROC-AUC для обеих моделей.

## Используемые данные

* **Файл:** `high_popularity_spotify_data.csv`
* **Признаки:** `energy`, `tempo`, `danceability`, `loudness`, `liveness`, `valence`, `speechiness`, `instrumentalness`, `acousticness`, `duration_ms`.
* **Целевая переменная:** `track_popularity` (0-100).
* **Предобработка:** Все числовые признаки нормализованы с помощью `StandardScaler`.

В ходе выполнения работы были получены следующие результаты:
